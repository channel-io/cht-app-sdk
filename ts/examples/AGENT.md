# Examples Agent Guide

Use the example apps as working references, not as the complete contract.

## Which Example To Choose

- [basic](./basic/README.md)
  - best for command registration
  - best first stop for command-to-WAM flow
- [calendar](./calendar/README.md)
  - best for calendar booking flow
  - useful for WAM structure
  - useful for OAuth plus calendar shape
