# Contributing

Thank you for contributing to Channel App SDK.

1. Fork the repository and create a focused branch.
2. Add tests and a Changeset for user-visible TypeScript package changes.
3. Run `make verify` before opening a pull request.
4. Describe public behavior without including private service names, internal
   issue IDs, credentials, customer data, or non-public URLs.

By submitting a contribution, you agree that it is licensed under the MIT
License included in this repository.

Security reports must follow [SECURITY.md](SECURITY.md) and must not be filed as
public issues.
